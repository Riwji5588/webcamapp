# Production Deployment Guide

## Required Files for Server:
- server/index.js
- public/index.html
- public/view.html
- public/obs.html
- package.json
- README.md

## Not needed:
- node_modules/ (will be installed on server)
- .git/ (if any)
- development files

## Environment Variables (optional):
- PORT=3000 (default)
- NODE_ENV=production
